# ptrcktylr.com

Personal website for Patrick Taylor. Minimal, NYC-inspired aesthetic. Professional portfolio meets personal creative space.

## About

A clean, minimal site showcasing tech projects with room to grow into art, music, and psychology resources. Built with vanilla HTML, CSS, and JavaScript. No frameworks, no build process, just clean code.

## Local Development

Open `index.html` in your browser, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000

## Structure


- Minimal, muted color palette inspired by NYC
- Clean typography, generous whitespace
- Professional but authentic
- Room to grow beyond tech into art, music, psych

## Deployment

This is a static site and can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

Just point the hosting service at the root directory.

## Technologies

- CSS3 (CSS Grid, Flexbox)

- System fonts for fast loading
